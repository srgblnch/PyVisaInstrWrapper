The development of this device server is in the PyVisaInstrWrapper, but it can be used in a stand alone mode as it is.

The series of devices under the PyVisaInstrWrapper name are design to connect to a PyVisaDS device who is responsble for the correct comunication to the instrument. Nowadays, this PyVisaDS is also in sourceforge, in tango_ds/Communication/PyVisaDS. Setting a property to this communications device, you have the minimum setup to start working.
